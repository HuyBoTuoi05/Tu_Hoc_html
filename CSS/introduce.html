CSS cơ bản

Attribute (thuộc tính)

            color: màu chữ 
            font-style: kiểu chữ
            font-family: font chữ        
            text-transform: chữ thường-hoa     
            text-align: căn chỉnh văn bản     
            font-size: Cỡ chữ    
            opacity: độ mờ chữ    
            display: hiển thị   
            visibility: ẩn đối tượng    
            background-color: màu nền của phân tử

Favicon (biểu tượng)

            Dùng favicon để tạo biểu tượng đại diện cho trang web của bạn. 
            Cách dùng: khai báo trong thẻ <head> </head>
            <link rel="shortcut icon" href="đường dẫn đến hình ảnh bạn muốn đặt làm icon" type="image/x-icon">

Prefix (sự tương thích với các trang web)

            Khi code CSS, sẽ có các thuộc tính không tương thích với một vài trình duyệt khác nhau, hoặc trình duyệt không hỗ trợ thuộc tính đó, ta sẽ dùng tiền tố prefix.
            Có thể copy code CSS, trang web sẽ tự thêm prefix cho những thuộc tính cần thiết.
            Link trang web: https://autoprefixer.github.io/
            Trang web kiểm tra thuộc tính CSS có cần sử dụng thêm prefix để hỗ trợ đa trình duyệt: https://caniuse.com/

Selector

1. Tag name selector (cơ bản -> tự học)

2. ID selector

            Khi chỉ muốn áp dụng CSS cho một phần tử chỉ định nào đó của HTML -> dùng ID.
            VD:
                Code trong HTML:
                            
                    <h2 id="II">Selector</h2>
                
                Code trong CSS:    
                
                    #II{
                        color: rgb(232, 7, 7);
                    }
                
                
            ID là duy nhất, theo quy chuẩn không được phép đặt trùng ID đã có.
            Trang web kiểm tra ID hợp lệ: https://validator.w3.org/#validate_by_input
            ID, class không được bắt đầu bằng số, dùng kí tự '_' hoặc '-' để kết nối nhiều từ.

3. Class selector

            Do không thể khai báo lặp lại bằng ID, nên khi muốn khai báo lặp ta dùng class.
            VD:
                Code trong HTML:
                            
                    <h2 class="header2">Selector</h2>
                
                Code trong CSS:    
            
                    .header2{
                        color: rgb(232, 7, 7);
                    }

4. * Selector (Chọn tất cả thuộc tính)

            VD: 
                Code trong HTML:
            
                    *{
                        color: rgb(232, 7, 7);
                    }

5. Element.class

            p.intro :
            Chọn tất cả các thẻ <p> với lớp "intro".

6. Attribute selector

            Chọn tất cả các thẻ có thuộc tính được chọn.
            VD: Chọn tất cả các thẻ có thuộc tính href.
                
                    [href]{
                        font-family: 'Courier New', Courier, monospace;
                        font-size: 15px;
                    }
                
                Chọn tất cả các thẻ có thuộc tính target="_self"
                
                    [target="_self"] {
                        font-family: 'Courier New', Courier, monospace;
                        font-size: 15px;
                    }
